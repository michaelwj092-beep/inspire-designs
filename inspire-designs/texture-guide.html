üíª Implementation Code

Body Element - Global Background

body {
    /* GLOBAL TEXTURED BACKGROUND - Acts as the "wall" behind all content */
    background-image:
        url('images/concrete-texture.jpg'),
        linear-gradient(135deg,
            #1a1a2e 0%,
            #16213e 25%,
            #1e1b4b 50%,
            #312e81 75%,
            #0a0a14 100%);
    
    background-size: auto, 400% 400%;
    background-position: center, 0% 50%;
    background-repeat: repeat, no-repeat;
    background-attachment: fixed;
    
    /* Blend the texture with the gradient */
    background-blend-mode: multiply;
    
    /* Subtle animation */
    animation: gradientShift 20s ease infinite;
}

Content Container Utility

.content-container {
