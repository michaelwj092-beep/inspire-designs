# Parallax Images

This folder contains images used for the **parallax background effect** on the website.

## Usage
- Images are displayed in 4 vertical columns with different scroll speeds
- Controlled by `parallax.js`
- Referenced in `config.js` under the portfolio categories

## Image Requirements
- **Format**: JPG or PNG
- **Recommended Size**: 800-1200px width for optimal performance
- **Aspect Ratio**: Vertical or square images work best
- Images will be displayed at full opacity with vibrant colors

## Current Configuration
The parallax system pulls images from the portfolio categories defined in `config.js`. To use images specifically for parallax, update the parallax.js file to reference this folder instead.
